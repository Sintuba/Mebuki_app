## プレフィックス

| プレフィックス | 用途 |
|---|---|
| `feat:` | 新機能 |
| `fix:` | バグ修正 |
| `refactor:` | リファクタリング |
| `docs:` | ドキュメント |
| `chore:` | その他（依存関係更新など） |

## 形式

```
{プレフィックス} {簡潔な説明}

{詳細な説明（任意）}
```

## 例

```
feat: ノート一覧にソート機能を追加

- 新しい順、古い順、タイトル順に対応
- ステータスフィルターと組み合わせ可能
```
