## v3 → v4 の主な変更点

- `tailwind.config.js` が不要になった（CSS ファイルで設定）
- `@import "tailwindcss"` で読み込む
- カスタムテーマは CSS 変数で定義

## CSS ベースの設定

```css
@import "tailwindcss";

@theme {
  --color-brand: #22c55e;
  --font-size-xs: 0.75rem;
}
```

## 気になる点

- `border-border` のような CSS変数参照クラスが使えるようになった
- `bg-background` / `text-foreground` で自動ダークモード対応
- Vite プラグインとの統合が改善された
